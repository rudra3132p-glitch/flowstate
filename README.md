## 🌊 About

**FlowState** is an intelligent workflow automation platform that combines a Python backend for data processing and machine learning with a modern Next.js frontend. It provides an intuitive interface for building, managing, and monitoring automated workflows.

> Automate your workflows with intelligence. Built with Python + Next.js.

<br/>

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔄 **Visual Workflow Builder** | Drag-and-drop interface for creating automation flows |
| 🤖 **ML Pipeline** | Integrated machine learning pipeline with Python backend |
| 📊 **Real-Time Monitoring** | Live dashboard tracking workflow execution and metrics |
| 🗄️ **Database Integration** | Built-in database layer for persistent workflow state |
| ⚡ **Fast Frontend** | Server-side rendered UI with Next.js for instant load times |
| 🔌 **Extensible Architecture** | Plugin-based system for adding custom workflow steps |
| 📈 **Analytics** | Workflow performance analytics and optimization suggestions |
| 🔐 **Secure Execution** | Sandboxed workflow execution with dependency isolation |

<br/>

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.10+

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rudra3132p-glitch/flowstate.git
   cd flowstate
   ```

2. **Install frontend dependencies**
   ```bash
   npm install
   ```

3. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Start the development servers**
   ```bash
   # Frontend
   npm run dev

   # Backend (in a separate terminal)
   python main.py
   ```

5. **Open** `http://localhost:3000` in your browser.
